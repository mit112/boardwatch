# boardwatch

Self-hosted job radar for technical job seekers: point it at the companies you
care about and it watches their **official ATS job boards** (no scraping, no
accounts, no API keys), normalizes new postings, scores them against your
profile with an explainable score, and hands you a shortlist.

**Status: pre-release, under active development.** No auto-apply, ever.
No telemetry. Local-first: your data stays in a local SQLite file.

See [docs/configuration.md](docs/configuration.md) for the supported settings keys and ranges.
