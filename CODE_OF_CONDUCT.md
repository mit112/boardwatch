# Code of Conduct

This project follows the spirit of the
[Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

Be respectful and constructive. Harassment, personal attacks, and
discriminatory language are not tolerated in any of this project's spaces
(issues, PRs, discussions, commits). Report unacceptable behavior privately to
the maintainer via GitHub; reports are handled confidentially. Maintainers may
edit or remove comments, commits, issues, and PRs that violate this standard,
and may ban repeat offenders.
